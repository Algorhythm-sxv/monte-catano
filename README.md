# Monte Catano: A High-Performance MCTS Engine for Catan

## Overview
Monte Catano is a **free MCTS engine for Catan** that analyzes positions and computes the optimal actions. <sub>Optimality not guaranteed</sub>
Monte Catano **does not include a GUI** to input actions, instead providing a CLI interface modeled after UCI (Unoversal Chess Interface) used by chess engines.

## Key features
- UCI-like interface to allow human and programmatic interaction via the command line
- Demo mode: watch the engine play a game against itself!
- **Built-in match runner**: Implements a Sequential Probability Ratio Test (SPRT) to determine if the engine is stronger than another (typically a previous version)
